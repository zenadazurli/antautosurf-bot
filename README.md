# AntAutoSurf Bot

Bot automatico per AntAutoSurf.com con supporto multi-account, proxy e headless mode.

## 🚀 Deploy su Render

1. **Fork** questo repository
2. **Connetti** a Render
3. **Imposta** le variabili d'ambiente:
   - `EMAIL` = tua_email@gmail.com
   - `PASSWORD` = tua_password
   - `HEADLESS` = true
   - `PROXY_HOST` = (opzionale)
   - `PROXY_PORT` = (opzionale)
   - `PROXY_USER` = (opzionale)
   - `PROXY_PASS` = (opzionale)
4. **Deploy**!

## 📌 Variabili d'ambiente

| Variabile | Obbligatoria | Default |
|-----------|--------------|---------|
| EMAIL | ✅ | - |
| PASSWORD | ✅ | - |
| HEADLESS | ❌ | True |
| PROXY_HOST | ❌ | - |
| PROXY_PORT | ❌ | - |
| PROXY_USER | ❌ | - |
| PROXY_PASS | ❌ | - |
